This is a simple task tracker web app made using Spring Boot and Thymeleaf.  
It lets you add tasks with a title and description, view them in a list, and delete them when done.

---

## Features

- Add new tasks
- View existing tasks
- Delete tasks
- Simple UI using Thymeleaf

---

## Tech Used

- Java
- Spring Boot
- Thymeleaf
- Maven (or Gradle)
